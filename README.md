## Folder Structure

After creation, your project should look like this:

```
GCA-DOC/
  README.md
  node_modules/
  package.json
  public/
    index.html
    favicon.ico
  src/
    app/
      App.js
      App.css
      config.prod.json
      config.dev.json

    components/
      headerBody/
        headerBody.js

      headerNav/
        HeaderNav.js

      headerIcon.js
      headerLogo.js
      headerMenu.js

    pages/
      content.js
      footer.js
      header.js

    utils/
      bulma.css
      bulma.css.map

    index.css
    index.js
```

For the project to build, **these files must exist with exact filenames**:

* `public/index.html` is the page template;
* `src/index.js` is the JavaScript entry point.
